
# 排列

m*(m-1)*(m-2)*...*2*1

# 组合

n 个元素里取出 m 个的组合，可能性数量就是 n 个里取 m 个的排列数量，除以 m 个全排列 的数量，也就是 (n! / (n-m)!) / m!。

对于全组合而言，可能性为 2^n 种。例如，当 n=3 的时候，全组合包括了 8 种情况。 

这两点都可以用数学归纳法证明，有兴趣的话你可以自己尝试一下。
